# 超链接标签

<a href="##" target="_blank"></a>
1

注意： 超链接默认是在当前窗体中打开页面，可以通过 target="\_blank" 设置新窗口中打开。 注意：href 中写一个#和两个##都是表示空链接，一个#号会刷新页面，回到页面顶部，两个##号表示不跳转

# 锚点的使用

锚：

<a href="#跳转位置的id名"></a>
1

点：任意标签上给一个 id="名字"

注意：使用锚点的时候需要页面足够长，如果只有一屏，锚点看不出效果。

# base 标签

就是统一设置 a 标签的属性：最常用的就是设置新窗口中打开 target="\_blank"，_self表示在当前窗口中打开

<base target="_blank" />
1

注意：base 标签应该写在 head 标签中

# 路径

在网页中找到图片的过程称之为路径。----找图片在哪？
在写路径的过程中：
进入一个文件夹的表达方式： 文件夹名字/ 例如: images/a.jpg
出来一个文件夹的表达方式： ../ 例如： ../a.jpg
当找到对应的图片后，直接写图片的名字.
上一级路径：../图片名字
下一级路径：文件夹名字/图片名字

# 同级路径

直接写图片名字

注意：路径的关系其实就是找图片，那么可能会有多个文件夹层次 注意：找路径的时候一定是以代码文件本身出发，去找图片

# 绝对路径

绝对不会变的位置。特点：带盘符 C:、带网址
